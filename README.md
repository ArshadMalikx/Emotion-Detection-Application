## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   # Backend
   cd backend
   npm install
   
   # Frontend
   cd frontend
   npm install
   ```
3. Set up environment variables
4. Start the development servers:
   ```bash
   # Backend
   npm run dev
   
   # Frontend
   npm start
   ```
